III. Set zsh as shell

    2. ~/.config/polybar  ⇒  ${DOTFILES_DIR}/polybar
    2. ~/.config/ranger  ⇒  ${DOTFILES_DIR}/ranger
    2. ~/.config/terminator  ⇒  ${DOTFILES_DIR}/terminator
